Etch-A-Sketch

Etch-A-Sketch is another exercise in DOM manipulation and vanilla Javascript.

As the app is loaded, I create a 64 x 64 grid of DIV "pixel" elements and use the addEventListener to watch for any occurrence when the user's mouse goes over each pixel.

As the event is triggered, each pixel's background is styled with whatever color value the color picker currently holds.

The Classic Calculator App

This calculator app uses transpiled Typescript written in an Object Oriented Programming (OOP) format.

There are two objects; a "Values" object and a "Calculation" object.

The Values object simply provides a container to store:

  • the values shown in the display
  • the first operand (or, the result of the previous calculation)
  • a boolean deciding if the app needs a second operand
  • the second (or next) operand
  • the operator

The Calculation object:

  • listens for user button clicks
  • updates the display
  • handles decimals
  • decides if it is ready to perform a calculation (if it needs a next operand)
  • performs the calculation
  • and then resets the app to default values at the user's command (AC)

A Newsletter Signup Page

After making a couple of foundational apps using simple HTML and CSS on the frontend and Node.js/Express.js on the backend, very basic calculators, I moved on to a more complex app that integrates a 3rd party API from Mailchimp .

This app sends an http POST request to a Mailchimp backend url, adding the user's form data to the contacts list in a pre-defined "Audience".

Then, based on the response status code, the user is redirected to a success or failure page.
